Description: Parameterized constructor for the lowPass derived filter class. This constructor uses the stepSize and f parameters to perform the more accurate calculation of alpha for the lowPass filter method.

Parameters
stepSizeis the change in time for each increment of the lowPass filter iteration.
fis the frequency for each increment of the lowPass filter iteration.
i_stateis the initial state of the system. This assumes that the state can always be represented as a vector.
Postcondition
Initializes a lowPass filter with an alpha value and initial state that filters a noisy signal for a given state.
